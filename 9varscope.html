Function scope

Function scope adalah aturan dalam JavaScript yang menentukan di mana variabel yang dideklarasikan di dalam function dapat diakses. Pada dasarnya, variabel yang dideklarasikan dengan var, let, atau const hanya dapat diakses di dalam function tempat variabel itu dideklarasikan.

Ciri-Ciri Function Scope:
Variabel yang dideklarasikan dalam sebuah function hanya tersedia di dalam function itu sendiri.
Variabel tersebut tidak bisa diakses dari luar function (bersifat lokal).
Function scope berlaku pada kata kunci var, let, dan const.
Contoh Function Scope

        function contohScope() {
            let pesan = "Halo dari dalam function!";
            console.log(pesan); // Bisa diakses di sini
        }

        contohScope(); // Output: Halo dari dalam function!
        console.log(pesan); // Error: pesan is not defined

Pada contoh ini:

Variabel pesan hanya bisa diakses di dalam function contohScope.
Jika mencoba mengakses pesan di luar function, akan terjadi ReferenceError karena pesan tidak ada di luar function.
Perbedaan Function Scope dan Global Scope
Global Scope:

Variabel dideklarasikan di luar function.
Dapat diakses di mana saja dalam kode.

        let globalVar = "Ini variabel global";

        function contohGlobal() {
            console.log(globalVar); // Bisa diakses di sini
        }

        contohGlobal(); // Output: Ini variabel global
        console.log(globalVar); // Output: Ini variabel global

Function Scope:

Variabel hanya hidup di dalam function tempat ia dideklarasikan.

        function contohFunctionScope() {
            let lokalVar = "Ini variabel lokal";
            console.log(lokalVar); // Bisa diakses di sini
        }

        contohFunctionScope(); // Output: Ini variabel lokal
        console.log(lokalVar); // Error: lokalVar is not defined

Hoisting dalam Function Scope

Khusus untuk variabel yang menggunakan var, ada mekanisme hoisting (pengangkatan deklarasi ke atas). Namun, let dan const tidak dihoist ke atas.

Contoh Hoisting dengan var:

        function hoistingDemo() {
            console.log(x); // Output: undefined (karena deklarasi diangkat)
            var x = 10;
            console.log(x); // Output: 10
        }

        hoistingDemo();

Contoh Tanpa Hoisting dengan let atau const:

        function tanpaHoisting() {
            console.log(y); // Error: Cannot access 'y' before initialization
            let y = 20;
        }
Fungsi Bersarang (Nested Function)
Variabel function scope juga bisa diakses oleh nested function (function di dalam function).

Contoh:

        function luar() {
            let pesan = "Hai dari luar!";
            
            function dalam() {
                console.log(pesan); // Bisa mengakses variabel di function 'luar'
            }

            dalam();
        }

        luar(); // Output: Hai dari luar!
        
Kesimpulan
Function Scope: Variabel hanya dapat diakses di dalam function tempat ia dideklarasikan.
Variabel global bisa diakses oleh function, tetapi variabel lokal tidak bisa diakses di luar function.
Gunakan function scope untuk menghindari konflik nama variabel dan menjaga modularitas kode.
