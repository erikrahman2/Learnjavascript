Di JavaScript, function adalah objek yang dapat dipanggil untuk mengeksekusi serangkaian perintah. Function didefinisikan sekali, tetapi bisa digunakan berkali-kali.

        Ciri utama function:
        
        1. Menerima input (parameter).
        2. Melakukan proses tertentu (logika).
        3. Mengembalikan hasil (return value) (opsional).

        Tujuan Function
        1. Reusability: Fungsi yang sama bisa digunakan berulang kali tanpa menulis ulang kodenya.
        2. Modularitas: Membagi program besar menjadi bagian-bagian kecil yang terpisah.
        3. Efisiensi: Membuat kode lebih ringkas dan lebih mudah dibaca.
        4. Maintainability: Jika ada perubahan, kamu cukup memperbarui function, dan semua pemanggilnya otomatis mengikuti.
        
        sintaks:

    sintaks function declaration:

        function namaFungsi(parameter) {
            // Logika
            return hasil; // (Opsional)
        }

        contoh:

        function tambah(a, b) {
            return a + b;
        }
        
        console.log(tambah(3, 4)); // Output: 7
        
    
        Built-in function adalah function bawaan yang sudah disediakan oleh JavaScript. Kamu tinggal memanggilnya, tanpa perlu mendefinisikannya sendiri.

Ciri-ciri Built-in Function:
Sudah ada secara default di JavaScript.
Bisa langsung digunakan tanpa perlu deklarasi.
Menyediakan berbagai fungsi umum yang sering digunakan, seperti manipulasi string, array, objek, atau angka.
    
        User-defined function adalah function yang kamu buat sendiri sesuai kebutuhan. Kamu mendefinisikan logika dan proses di dalamnya.

Ciri-ciri User-defined Function:
Dibuat manual oleh programmer untuk tujuan tertentu.
Memiliki nama, parameter (opsional), dan logika.
Digunakan untuk menyelesaikan masalah spesifik yang tidak disediakan oleh built-in function.
