DOM manipulation

DOM Manipulation adalah cara kita mengubah elemen HTML dengan JavaScript secara dinamis. Dengan ini, kita bisa menambah, menghapus, atau mengubah elemen dan isinya tanpa harus menyegarkan halaman.

Manipulasi Elemen (Element Nodes)

Elemen dalam DOM merujuk pada tag HTML yang dapat diubah atau dimanipulasi seperti div, p, button, dll. 

      let title = document.getElementById("judul"); // Ambil elemen dengan id
      let paragraphs = document.getElementsByTagName("p"); // Ambil semua elemen p
      let boxes = document.getElementsByClassName("box"); // Ambil elemen dengan class tertentu
      let button = document.querySelector("#myButton"); // Ambil elemen pertama yang cocok dengan selector
      let allItems = document.querySelectorAll(".item"); // Ambil semua elemen dengan class "item"

Manipulasi Node (Semua Jenis Node dalam DOM)

Selain elemen, DOM juga memiliki node lain, seperti text node, comment node, dan document node. 

      createElement() → Membuat elemen baru
      appendChild(), prepend() → Menambahkan elemen
      insertBefore(newElement, referenceElement) digunakan untuk menyisipkan 
      elemen sebelum elemen referensi di dalam suatu parent node.
      remove(), removeChild() → Menghapus elemen
      replaceChild() → Mengganti elemen
      cloneNode() → Menyalin elemen

Apa Itu Elemen? Elemen (Element Node) adalah bagian dari Node yang berupa tag HTML, seperti div, p, span, dll. Singkatnya: Semua Elemen adalah Node, tetapi Tidak Semua Node adalah Elemen (karena ada teks, komentar, dll.).
