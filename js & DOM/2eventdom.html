events

DOM Event adalah peristiwa yang terjadi pada elemen HTML, seperti klik tombol, mengetik di input, atau menggulir halaman. JavaScript memungkinkan kita untuk menangani event ini dengan berbagai cara.

cara menangani event:

kasus: event ketika kotak dibawah di klik akan berubah warna

  • event attribute -- inline html

    cara ini tidak fleksibel dan sangat tidak disarankan untuk big project

  • event handler -- javascript

    cara ini Hanya bisa satu event per elemen (menimpa event sebelumnya)

  • addeventlistener()

    cara ini sangat disarankan
