JavaScript DOM (Document Object Model)

adalah API yang memungkinkan JavaScript untuk berinteraksi dengan dan memanipulasi elemen HTML dan struktur dokumen web secara dinamis.

konsep dasar Document object model

DOM merepresentasikan halaman web dalam bentuk pohon dokumen, di mana setiap elemen adalah sebuah node yang bisa dimanipulasi menggunakan javascript

contoh struktur DOM untuk Html:

        !doctype html
        html 
        head
            title 'DOM Example' /title 
        /head
        body 
            h1 id= judul 'hello dom' /h1
            p class= text 'paragraf pertama' /p 
        /body 
        /html

hello dom

paragraf pertama

struktur DOMnya:

    document
    ├── html
    │   ├── head
    │   │   ├── title
    │   ├── body
    │       ├── h1#judul
    │       ├── p.text
