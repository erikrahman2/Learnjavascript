func. declaration

Function Declaration adalah cara mendefinisikan fungsi menggunakan kata kunci function, diikuti nama fungsi, parameter (jika ada), dan blok kode yang berisi logika fungsi. Fungsi ini bersifat hoisted, artinya dapat dipanggil sebelum dideklarasikan dalam kode. 

syntax:
        function namaFungsi(parameter1, parameter2) {
            // kode fungsi
            return hasil;
}

Karakteristik:
Dapat dipanggil di mana saja dalam scope karena hoisting.
Nama fungsi wajib ada.
Dibaca lebih awal oleh JavaScript sebelum eksekusi.

func. eXpression

Function Expression adalah cara mendefinisikan fungsi dengan menyimpannya ke dalam variabel. Fungsi ini tidak di-hoist, sehingga hanya dapat dipanggil setelah deklarasi variabel. 

syntax:
        const namaVariabel = function(parameter1, parameter2) {
            // kode fungsi
            return hasil;
        };

Karakteristik:
Tidak di-hoist, sehingga hanya dapat dipanggil setelah dideklarasikan.
Nama fungsi opsional (anonymous function).
Lebih fleksibel untuk digunakan sebagai nilai variabel, parameter, atau callback.

kapan menggunakan function diatas

Function Declaration:

Cocok untuk fungsi umum yang digunakan di berbagai tempat. Baik untuk mendefinisikan fungsi utama dalam program. Function Expression:
Digunakan saat fungsi hanya dibutuhkan pada konteks tertentu (misalnya sebagai callback). Berguna untuk mendeklarasikan fungsi lokal atau fungsi anonim.
