if else

      if-else adalah salah satu struktur kendali dalam JavaScript yang digunakan untuk membuat keputusan berdasarkan kondisi tertentu. Dengan kata lain, ini adalah cara program “memilih” jalur eksekusi kode berdasarkan apakah suatu pernyataan benar (true) atau salah (false).

Pengertian:
if: Memeriksa suatu kondisi. Jika kondisinya true, maka blok kode di dalamnya akan dijalankan.
else: Menyediakan alternatif jika kondisi if tidak terpenuhi (false).
else if: Digunakan untuk memeriksa lebih dari satu kondisi secara berurutan.
Contoh Ilustrasi:
Bayangkan kamu punya pintu otomatis di rumah.

Kondisi pertama: Kalau ada orang di depan pintu, pintu akan terbuka.
Kondisi kedua: Kalau tidak ada orang, pintu akan tetap tertutup.
Kondisi tambahan: Kalau sensor rusak, akan ada alarm yang berbunyi.
