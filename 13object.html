object

Object adalah struktur data yang digunakan untuk menyimpan koleksi nilai dalam bentuk key-value pairs. Mirip seperti array, tetapi bukan berbasis indeks, melainkan property (key). 

        1. Object menyimpan data dalam format key-value.
        2. Bisa dibuat dengan object literal, constructor, atau class.
        3. Memiliki method bawaan seperti keys(), values(), entries().
        4. Iterasi menggunakan for...in.
        5. Gunakan spread operator (...) dan destructuring untuk manipulasi lebih efisien.
        6. JSON digunakan untuk menyimpan dan bertukar data.
