switch

switch digunakan untuk mengevaluasi sebuah ekspresi dan mencocokkannya dengan beberapa case. Ketika cocok, blok kode dalam case tersebut akan dijalankan. Jika tidak ada yang cocok, default akan dijalankan (opsional). 

switch (ekspresi) {
        case nilai1:
            // Kode untuk nilai1
            break;
        case nilai2:
            // Kode untuk nilai2
            break;
        default:
            // Kode jika tidak ada yang cocok
    }

Cara Kerja Ekspresi dalam switch dievaluasi. Hasil ekspresi dibandingkan dengan setiap nilai case. Jika cocok, kode di dalam case tersebut dijalankan. break digunakan untuk menghentikan eksekusi setelah menemukan case yang cocok. Jika tidak ada nilai yang cocok, blok default akan dijalankan (jika ada).

Tanpa break (Perhatian!) Jika break dihilangkan, semua case di bawahnya akan dijalankan, meskipun tidak cocok.
