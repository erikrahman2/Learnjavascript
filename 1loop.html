Loop

digunakan untuk menjalankan blok kode berulang kali, tetapi cara kerjanya sedikit berbeda tergantung pada kasusnya.

while

Cocok digunakan jika kita tidak tahu pasti jumlah iterasi, tapi tahu kondisinya. 

while (kondisi) {
        // Blok kode yang akan dijalankan
    }

for

Digunakan ketika jumlah iterasi sudah diketahui sebelumnya. Cocok untuk situasi seperti iterasi array, penghitungan angka, dll. 

for (init; kondisi; increment/decrement) {
      // Blok kode
  }
